# linuxreport
Helper tool to check user linux status quick

Usage python main.py --host 192.168.1.10 --user myuser --password mypass

Run needed steps :
python main.py --host 192.168.1.10 --user myuser --password mypass --steps 1,3

or from certain step:
python main.py --host 192.168.1.10 --user myuser --password mypass --from-step 2

Help
python main.py --help
